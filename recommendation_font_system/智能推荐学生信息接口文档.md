## 智能推荐模块学生信息接口文档

### 1. 接口概述
- **接口地址**：`http://localhost:8080/recommendation/student/info`
- **前端请求地址**：`/api/recommendation/student/info`
- **请求方式**：`GET`
- **鉴权**：需要携带登录后获取的 `Authorization` Token

### 2. 请求参数
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| studentId | String | 是 | 学生编号 |

请求示例：
```
GET /recommendation/student/info?studentId=20230002
Authorization: Bearer {token}
```

### 3. 响应参数
| 字段 | 类型 | 说明 |
|------|------|------|
| code | Number | 业务状态码，200 表示成功 |
| message | String | 提示信息 |
| data | Object | 学生概要信息 |

`data` 字段说明：
| 字段 | 类型 | 说明 |
|------|------|------|
| studentId | String | 学生编号 |
| studentName | String | 学生姓名 |
| examScore | Number | 高考分数 |
| provinceRank | Number | 省市排名 |
| volunteerCount | Number | 已填报志愿数量 |
| admissionMatchCount | Number | 匹配的历史录取数据数量 |

成功响应示例：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "studentId": "20230002",
    "studentName": "张三",
    "examScore": 650.5,
    "provinceRank": 5000,
    "volunteerCount": 6,
    "admissionMatchCount": 3
  }
}
```

失败响应示例：
```json
{
  "code": 404,
  "message": "未找到对应学生信息",
  "data": null
}
```

### 4. 前端调用示例
```javascript
import { getRecommendationStudentInfo } from '@/api/recommendation'

async function fetchStudentSummary(studentId) {
  const res = await getRecommendationStudentInfo(studentId)
  if (res.code === 200 && res.data) {
    return res.data
  }
  throw new Error(res.message || '获取学生信息失败')
}

// 志愿推荐、冲突检测和综合分析页面都会在查询前调用该接口，
// 将返回的 examScore / volunteerCount / admissionMatchCount 等信息
// 渲染到“学生信息”卡片中，确保展示与后端数据实时同步。
```

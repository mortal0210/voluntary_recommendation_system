## 个人中心接口文档

### 接口概览
- **接口地址**：`http://localhost:8080/user/profile`
- **前端请求地址**：`/api/user/profile`
- **请求方法**：`GET`
- **是否需要鉴权**：是（需携带登录后获取的 `Authorization` token）

### 请求参数
无额外查询参数，token 通过请求头传递：
```
Authorization: Bearer {token}
```

### 响应参数
| 字段 | 类型 | 说明 |
|------|------|------|
| code | Number | 业务状态码，200 表示成功 |
| message | String | 操作描述 |
| data | Object | 个人中心数据 |

`data` 对象字段：
| 字段 | 类型 | 说明 |
|------|------|------|
| userId | Number | 用户编号 |
| username | String | 用户名 |
| role | String | 角色标识：`admin`、`common`、`student` |
| roleLabel | String | 角色名称（由前端计算，可选） |
| studentId | String | 学生编号（学生角色可用） |
| status | Number | 用户状态：1 启用，0 禁用 |
| createTime | String | 创建时间 |
| lastLoginTime | String | 最近登录时间 |
| phone | String | 联系方式 |
| lastAction | String | 最近操作描述 |
| department | String | 所属部门 |

### 成功响应示例
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "userId": 3,
    "username": "student2",
    "role": "student",
    "studentId": "20230002",
    "status": 1,
    "createTime": "2025-06-20 00:20:29",
    "lastLoginTime": "2025-12-02 10:00:00",
    "phone": "13800000000",
    "lastAction": "更新了个人资料",
    "department": "招生办"
  }
}
```

### 失败响应示例
```json
{
  "code": 401,
  "message": "登录已过期，请重新登录",
  "data": null
}
```

### 调用示例（前端）
```javascript
import { getProfileInfo } from '@/api/profile'

getProfileInfo().then(res => {
  if (res.code === 200) {
    console.log('个人信息', res.data)
  }
})
```
